# Hoofdstuk methoden
## Oefening Kostprijs berekenen
- Maak een methode `CalculateTotalPrice` op basis van de dagprijs en het aantal dagen de totaalprijs berekent.
- `BtnCalculateTotalPrice_Click`:
  - Lees de gekozen dagprijs en aantal dagen uit de GUI uit en bereken op basis van deze input de totaalprijs.
  - Toon het resultaat in `lblTotalPrice`
- `sldNumberOfDays`: bij elke wijziging in de waarde, wordt deze in `txtNumberOfDays` getoond.
