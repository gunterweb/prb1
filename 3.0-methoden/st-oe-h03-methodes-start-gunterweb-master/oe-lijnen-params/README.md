# Methoden met parameters

## Toevoeging methoden horizontale en verticale lijnen

### DrawHorizontalLine

Deze methode trekt een horizontale lijn in het linker canvas.
Dit op een bepaalde hoogte en voor een bepaalde lengte. De lijn begint altijd vanaf de linker marge.

Welke info heeft deze methode nodig om te kunnen werken? M.a.w. Welke parameters heeft deze methode nodig?

Schrijf deze methode zo efficiënt mogelijk op basis van de bestaande methodes.

Maak een call naar deze methode vanuit BtnDrawHorizontalLine_Click.

### DrawVerticalLine

Deze methode trekt een verticale lijn in het rechter canvas.
Dit op een bepaalde afstand van de linkermarge en voor een bepaalde lengte. De lijn begint altijd vanaf de bovenkant.

Welke info heeft deze methode nodig om te kunnen werken? M.a.w. Welke parameters heeft deze methode nodig?

Schrijf deze methode zo efficiënt mogelijk op basis van de bestaande methodes.

Maak een call naar deze methode vanuit BtnDrawVerticalLine_Click.

## Aanpassing

Tot hiertoe hadden de lijnen altijd een dikte van 5 pixels. We willen dit nu laten bepalen door de gebruiker van de methode.

Pas de header van de bovenstaande methodes aan, maar zorg ervoor dat er in de bestaande calls geen problemen komen.

Pas een call aan, zodat er random een lijndikte doorgegeven wordt variërend van 1 - 10 pixels.
